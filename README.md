# DocManager 使用说明

## 简介

DocManager 是一款桌面文档管理工具，帮助你快速归档和搜索学校访问方式的文档（VPN、邮箱、教务系统等）。

## 启动方式

```bash
cd D:\workplace\docmanager-v0
npm run dev
```

## 主要功能

### 1. 拖放导入文档

将 **txt / doc / docx / xls / xlsx** 文件直接拖入窗口中央区域，系统会自动：

- 解析文档内容（Word、Excel、纯文本）
- 根据内容关键字自动分类
- 生成内容摘要

### 2. 全文搜索

在顶部搜索框输入关键字（如 "VPN"、"密码"、"图书馆"），系统会：

- 实时搜索所有已导入文档的**内容**（不是文件名）
- 高亮显示匹配的关键字
- 显示匹配片段的上下文
- 按匹配数量排序结果

支持 200ms 防抖，输入即搜。按 `Escape` 清空搜索。

### 3. 分类筛选

左侧边栏显示所有分类，点击某个分类只显示对应文档：

| 分类 | 触发关键字 |
|------|-----------|
| VPN访问 | VPN、隧道、代理 |
| 邮箱 | email、mail、smtp、exchange |
| 教务系统 | 选课、成绩、学分、课程 |
| 图书馆 | 文献、论文、期刊 |
| 财务系统 | 报销、工资、经费 |
| OA办公 | 考勤、审批、公告 |
| 门户系统 | portal、统一认证、SSO |
| 一卡通 | 校园卡、门禁、消费 |
| 宿舍管理 | 公寓、住宿、寝室 |
| 其他文档 | 未匹配任何规则 |

### 4. 查看文档详情

点击任意搜索结果，右侧滑出预览面板，显示：

- 文件名、类型、分类、标签
- 文件大小、导入时间
- 文档完整内容

点击面板外部或按 `Escape` 关闭。

### 5. 删除文档

鼠标悬停在文档条目上，右侧出现「删除」按钮，点击确认后删除。

## 数据存储

所有数据存储在本地：

```
%APPDATA%/docmanager/app-data/documents.json
```

- 卸载应用不会丢失数据
- 如需备份，直接复制该 JSON 文件即可
- 如需清空，删除该文件后重启应用

## 快捷键

| 按键 | 功能 |
|------|------|
| `Escape` | 关闭预览面板 / 清空搜索 |

## 支持的文件格式

| 格式 | 扩展名 |
|------|--------|
| 纯文本 | .txt |
| Word 文档 | .doc, .docx |
| Excel 表格 | .xls, .xlsx |
