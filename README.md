# DocManager

桌面文档管理工具 — 拖放导入、自动分类、全文搜索。

## 快速开始

```bash
git clone https://github.com/Woodenrabbit/docmanager.git
cd docmanager
npm install
npm run dev
```

## 功能

### 导入文档
- **侧栏上传区**：侧栏底部「+ 导入文档」区域，支持拖放文件或点击打开系统文件选择器
- **独立导入页**：左侧活动栏 `+` 图标切换到全屏导入页面，支持拖放和选择文件
- 重复导入相同文件会自动更新已有记录，不会产生重复
- 支持格式：**txt / doc / docx / xls / xlsx**

### 全文搜索
顶部搜索框输入关键字，实时搜索所有文档内容，高亮匹配关键字，显示上下文片段。

### 自动分类

| 分类 | 匹配规则 |
|------|---------|
| VPN访问 | VPN、隧道、代理 |
| 邮箱 | email、mail、smtp、exchange |
| 教务系统 | 选课、成绩、学分、课程 |
| 图书馆 | 文献、论文、期刊 |
| 财务系统 | 报销、工资、经费 |
| OA办公 | 考勤、审批、公告 |
| 门户系统 | portal、统一认证、SSO |
| 其他文档 | 未匹配 |

> 可手动更改分类：右键文档 →「移动到分类」→ 选择已有分类或输入自定义分类名。

### 侧边栏
- 文件名搜索过滤
- 分类可折叠/展开（点击箭头）
- 点击分类名筛选该分类文档
- 拖拽分隔线调整侧栏宽度
- 底部上传区：拖放或点击导入文档

### 预览面板
- 点击文件标题 → 右侧滑出完整内容
- 点击匹配片段 → 打开预览并自动滚动到关键字位置，高亮所有匹配

### 右键菜单
- 用默认程序打开原始文件
- 更改分类（选择已有 / 输入自定义）
- 勾选/取消标签（登录凭证、网址链接、联系方式、操作指南）

### 打包导出
标题栏 **📦 打包导出** 按钮 — 将所有文档按分类打包为 ZIP 下载。

## 快捷键

| 按键 | 功能 |
|------|------|
| `Escape` | 关闭预览面板 / 清空搜索 |
